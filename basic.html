HTML 基礎教學：給大家的入門講義

從「看得懂」到「自己做得出來」的結構化練習

對於國中的學生來說，學習 HTML 的重點不在於記憶所有的標籤，而在於建立「結構化思維」。 你可以將網頁想像成一棟房子：HTML 是骨架（樑柱）、CSS 是裝潢（油漆）、JavaScript 則是家具的功能（開關燈）。

針對國二學生的認知程度，建議教學內容可以分為以下五個階段：

1. 基礎觀念：網頁是怎麼構成的？

在動手寫程式碼之前，先讓理解網頁的運作邏輯：

  • 瀏覽器與檔案：瀏覽器（Chrome, Edge）是用來「解讀」HTML 檔案的工具。
  • 標籤的概念：HTML 是由一對對的「標籤」（Tags）組成的，就像是括號一樣，告訴電腦中間的內容是什麼。
    例如：<標籤> 內容 </標籤>（有頭有尾，結尾多一個斜線 /）。

2. 認識標準結構（骨架）

讓學生看一段最標準的 HTML 程式碼，並解釋每個部分的作用。這對他們來說就像是寫信的格式：

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>我的第一個網頁</title>
  </head>
  <body>
  </body>
</html>
  • <html>：整個網頁的範圍。
  • <head>：網頁的「腦袋」，放標題、編碼等不直接顯示在畫面上的資訊。
  • <body>：網頁的「身體」，所有你在瀏覽器上看到的文字、圖片都放在這裡。

3. 必學的基礎標籤（常用語法）

這部分可以讓學生實際操作，練習更改文字內容並觀察畫面變化：

A. 標題與段落

  • <h1> 到 <h6>：標題標籤。<h1> 是最大標題，<h6> 是最小標題。
  • <p>：段落（Paragraph），用來放一般文字內容。

B. 條列清單

  • <ul> 與 <li>：無序清單（點點符號）。
  • <ol> 與 <li>：有序清單（1, 2, 3 數字編號）。

C. 多媒體與連結

  • <a>：超連結。重點在於 href 屬性（要去哪裡）。
  • <img>：圖片。重點在於 src（圖片路徑）。這是少數「單邊」不需要結尾標籤的例子。

4. 基礎 CSS：讓網頁變好看

CSS 就像是幫網頁「穿衣服」，可以改顏色、字體和間距。我們先教學生用最簡單的方式：直接把 CSS 寫在 <style> 裡面，不用分成兩個檔案。

A. 在同一個檔案裡寫 CSS

請同學在 <head> 裡面加上 <style> 區塊，例如：

<head>
  <meta charset="UTF-8">
  <title>我的第一個網頁</title>
  <style>
    body {
      background-color: #fef3c7;
      font-family: "Microsoft JhengHei", sans-serif;
    }
    h1 {
      color: #b91c1c;
      text-align: center;
    }
    p {
      line-height: 1.8;
    }
  </style>
</head>

B. 說明給學生聽

  • background-color：設定背景顏色，例如 background-color: #fef3c7;。
  • font-family：設定字型，例如 font-family: "Microsoft JhengHei", sans-serif;。
  • color：設定文字顏色，例如 color: #b91c1c;。
  • text-align：設定文字對齊方式，例如 text-align: center; 讓標題置中。
  • line-height：設定行距，例如 line-height: 1.8; 讓段落文字比較好讀。

C. 小小挑戰

  • 請學生把自己的「自我介紹」網頁背景顏色改成自己喜歡的顏色。
  • 把自己的名字標題改成不同顏色，並置中顯示。
  • 調整段落行距，讓文字不要太擠。

5. 互動練習：做一張「自我介紹」網頁

與其死背語法，不如讓學生直接完成一個小作品。內容可以包含：

  • 1. 大標題：我的姓名。
  • 2. 小標題：我的興趣。
  • 3. 清單：列出三個喜歡的食物。
  • 4. 圖片：放一張喜歡的偶像或動漫角色。
  • 5. 連結：連到學校官網或自己的社群帳號。
老師小提醒：可以請學生將這份「自我介紹網頁」命名為 me.html， 並存放在 www 資料夾中，後續再用主選單頁面把它串連起來。
